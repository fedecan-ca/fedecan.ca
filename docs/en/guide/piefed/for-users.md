# Piefed: Guides for Users

## What is PieFed?

PieFed is a forum-style social media platform, often referred to as a 'link aggregator'. You may be familiar with websites such as Reddit or Hacker News, which are similar in nature. You can share and discuss links, text, and images, and decide on what content rises to the top by upvoting and downvoting.

PieFed is similar to other forum-style websites in that users make posts to communities that are centered on a theme or topic.

**PieFed is a part of the 'fediverse'.** This means that you can access communities from all over the internet, as if they were all on the same website.

If you want to learn about this in detail, you can see the [Detailed Overview](../lemmy/for-users/detailed-overview.md) page from the lemmy guide. All the same principles work the same for PieFed. However, if you're ready to get started, see below!

## Getting Started with PieFed in 3 Steps

---

### 1. Pick an Instance

When you made an email address, you got to pick which website you made it with. You can do the same with PieFed! PieFed includes a built-in [instance chooser](https://piefed.social/auth/instance_chooser) tool that can be enabled to help browse and compare different PieFed instances.

::: tip üç¶ Can't decide?

If you are having a hard time picking, you're not alone. With so many options, you might feel like you're at an ice cream shop, unable to pick which flavor to try.

Remember that you can always make an account in one place, and then move somewhere else later. Some people even have accounts on a few places at the same time.
:::

The instance that we run is [piefed.ca](https://piefed.ca), which is geared towards Canadians, but open to everyone.

---

### 2. Make an Account

This part is easy!

As mentioned above, we run **piefed.ca**, which you can join by going to [piefed.ca/auth/register](https://piefed.ca/auth/register). If you want to join a different instance, look for a 'register' button on their website.

In order to reduce spam, some instances will have a sign up form. After you submit, an admin will review your account and approve it manually. This can take a few hours.

---

### 3. Start Posting

Congratulations! üéâ You are now free to use PieFed however you like.

To make the most of your experience:

- Join the [!newcomers@piefed.zip](https://piefed.zip/c/newcomers) community. This community is specifically for new users, and you can come back to it whenever you have questions.
- Grab a mobile app or install PieFed as a progressive web application (PWA) on your phone. <!--You can find a list of mobile apps [here](/en/guide/lemmy/for-users/mobile-apps).-->
