# Pixelfed Guides

Every Pixelfed instance has a help section with guides on how to use the platform. You can find these guides by using the "Help" link at the bottom of the sidebar.

For example, you can access the guides for [pixelfed.ca](https://pixelfed.ca) by visiting [pixelfed.ca/site/help](https://pixelfed.ca/site/help).

## Additional Guides

We may add additional guides here in the future, but for now, we recommend checking out the guides on the instance you're using.
